📅 2026-05-28 📁 环境部署 Rocky Linux VMware 虚拟化

预先准备

在开始安装之前，需要准备好以下软件和镜像文件：

下载 Rocky Linux 10 镜像

完整版 DVD：https://download.rockylinux.org/pub/rocky/10/isos/x86_64/Rocky-10.0-x86_64-dvd.iso
精简版 Minimal：https://download.rockylinux.org/pub/rocky/10/isos/x86_64/Rocky-10.0-x86_64-minimal.iso

验证

完成以上步骤后，请验证以下几点确保安装成功：

验证项 预期结果 验证命令
网络连通性 能 ping 通外网 ping -c 3 www.baidu.com
IP 地址 获取到有效 IP ip addr show ens160
SSH 服务 SSH 端口开放 ss -tlnp | grep :22
Root 登录 可通过 SSH 使用 Root 登录 在 MobaXterm 中测试连接

注意事项

  • 磁盘容量：创建的虚拟机磁盘是虚拟的，实际不占用物理空间，但会影响性能
  • 内存分配：内存越大，虚拟机运行越流畅，建议分配 4GB 或更多
  • 虚拟化支持：确保在 BIOS 中启用了 CPU 虚拟化功能（Intel VT-x / AMD-V）
  • 网络模式：默认使用 NAT 模式，虚拟机可通过宿主机访问外网
  • 快照功能：安装完成后建议创建快照，方便后续出现问题时快速恢复
