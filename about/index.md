# About SRE Wiki

这是一个 SRE (Site Reliability Engineering) 运行手册和文档站点。

## 目标

- 📚 积累和分享运维经验
- 🔧 提供标准化的故障处理流程
- 📖 帮助团队快速定位和解决问题

## 内容分类

| 分类 | 说明 |
|------|------|
| Runbook | 具体的故障处理手册 |
| SOP | 标准操作流程 |
| Reference | 常用命令和配置参考 |

## 贡献指南

1. 在 `source/_posts/` 目录下创建 Markdown 文件
2. 按照模板格式编写内容
3. 运行 `npx hexo clean ; npx hexo deploy` 发布
