K8s Pod 调试手册

📅 2026-05-25 📁 运维手册 kubernetes k8s debug pod

快速检查

# Pod 状态
kubectl get pods -n &lt;namespace&gt;

# Pod 事件
kubectl describe pod &lt;pod-name&gt; -n &lt;namespace&gt;

# Pod 日志
kubectl logs -f &lt;pod-name&gt; -n &lt;namespace&gt; --tail=100

重启问题

# 检查重启次数
kubectl get pods | grep -v Running

# 检查存活探针/就绪探针
kubectl describe pod &lt;pod-name&gt; | grep -A5 "Liveness\|Readiness"

资源问题

# Pod 资源使用情况
kubectl top pod -n &lt;namespace&gt;

# 节点容量
kubectl describe nodes | grep -A5 "Allocated resources"

网络调试

# Pod IP
kubectl get pod -o wide

# Service 端点
kubectl get endpoints &lt;svc-name&gt; -n &lt;namespace&gt;

# DNS 解析测试
kubectl run dnsutils --rm -it --image=tutum/dnsutils --restart=Never -- nslookup &lt;service-name&gt;

进入 Pod Shell

kubectl exec -it &lt;pod-name&gt; -n &lt;namespace&gt; -- /bin/sh

常见修复

# 强制重启 Pod
kubectl delete pod &lt;pod-name&gt; -n &lt;namespace&gt;

# 扩容
kubectl scale deployment &lt;deploy-name&gt; --replicas=3 -n &lt;namespace&gt;
